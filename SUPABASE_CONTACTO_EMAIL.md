# Contacto: Guardar en Supabase y Enviar Correo

Este proyecto tiene el formulario de Contacto implementado para llamar una Supabase Edge Function llamada `contacto`. Esa function:

- Inserta el mensaje en `public.lex_contactos`
- Envía un correo con Resend al destinatario configurado

## 1) Crear tabla (Supabase)

En Supabase: **SQL Editor**, ejecuta:

```sql
create table if not exists public.lex_contactos (
  id bigint generated by default as identity primary key,
  created_at timestamptz not null default now(),
  nombre text not null,
  email text not null,
  asunto text not null,
  mensaje text not null,
  origen text
);

alter table public.lex_contactos enable row level security;
```

Nota: La Edge Function inserta usando `service_role`, asi que no necesitas policies de `insert` para `anon`.

## 2) Crear cuenta y API Key (Resend)

En Resend:

- Crea un API key (lo usaremos como `RESEND_API_KEY`).
- Define el remitente (`from`). Para produccion necesitas un dominio verificado. Para pruebas puedes intentar con `onboarding@resend.dev` (puede tener limitaciones).

## 3) Crear y desplegar la Edge Function (Supabase)

El codigo esta en:

- `supabase/functions/contacto/index.ts`
- `supabase/functions/contacto/config.toml` (con `verify_jwt = false`)

Opciones para desplegar:

- **Con Supabase CLI**: `supabase functions deploy contacto`
- **Sin CLI**: copia el contenido de `index.ts` en tu function `contacto` dentro del Dashboard (si tu plan/UX lo permite).

## 4) Configurar Secrets (Supabase)

En Supabase: **Edge Functions** -> **Secrets**, agrega:

- `SUPABASE_URL`: URL del proyecto (Settings -> API)
- `SUPABASE_SERVICE_ROLE_KEY`: Service role key (Settings -> API)
- `RESEND_API_KEY`: API key de Resend
- `CONTACT_TO_EMAIL`: `erhos1402@gmail.com`
- `RESEND_FROM_EMAIL`: por ejemplo `Lex Corporativa <onboarding@resend.dev>` o tu dominio verificado

## 5) Probar

En el front, el submit llama:

- `supabase.functions.invoke('contacto', { body: { nombre, email, asunto, mensaje, origen } })`

Si todo esta bien:

- Veras el registro en `public.lex_contactos`
- Llegara el correo a `erhos1402@gmail.com`
